# Juiccy - An e-commerce website
## Description
This project is based on e-commerce. It's a full stack web development project. It aims to create a responsive and dynamic webapp/website. This website offers a wide range of features and functionalities to enhance the user experience. Users can browse through a diverse selection of beverages, coffee, etc. and can add their favourite items in the cart along with the quantity needed. The website also allows users to see details of each item along with the ratings. Currently data of items are static.
